The **Agent Development Kit (ADK)** is an open-source, code-first framework designed to build, evaluate, and deploy agentic workflows.
It provides a graph-based workflow runtime supporting sequential execution, routing, loops, and parallel tasks.
Additionally, it supports Human-in-the-Loop (HITL), memory management, and LiteLLM model routing.
